# 🔐 Password Strength Checker v1

*A beginner-friendly Python tool to check password strength and give improvement tips*

---

## ⭐ Features

- ✅ Checks password length
- ✅ Validates uppercase, lowercase, digits & special characters  
- ✅ Blocks common weak passwords
- ✅ Gives a score from 0-4 and tells you what's missing
- ✅ Color-coded output for easy reading

## 🚀 How to Run

```bash
git clone https://github.com/hirushi-cybersec/cybersecurity-portfolio.git
