## Features
- *Breach Check*: Uses Have I Been Pwned API to check if password was leaked
- *Crack Time Estimate*: Calculates time to crack using brute force attacks  
- *Star Rating*: Visual strength rating from 1 to 5 stars ⭐
- *Smart Suggestions*: Gives specific tips to make password stronger

## Star Rating System
- ★☆ *Very Weak* - 0-1 points
- ★★☆ *Weak* - 2-3 points  
- ★★★☆ *Medium* - 4-5 points
- ★★★★☆ *Strong* - 6 points
- ★★★ *Very Strong* - 7 points
